## Overview
This project is a responsive and accessible profile card built with HTML, CSS, and vanilla JavaScript.

It presents a clean personal card layout with a live epoch time display, semantic markup, and keyboard-friendly social links.

---

## Features

- Displays user name, bio, avatar, hobbies, and dislikes
- Shows the current epoch time in milliseconds using `Date.now()`
- Updates the time every second with `setInterval`
- Uses a lazy-loaded profile image
- Opens social links safely in a new tab
- Uses semantic HTML elements such as `article`, `header`, `nav`, and `section`
- Adapts cleanly across mobile, tablet, and desktop layouts
- Includes visible keyboard focus styles

---

## Accessibility

- Meaningful `alt` text on the avatar image
- Keyboard-accessible social links
- `aria-live="polite"` on the time output
- Strong contrast and readable spacing
